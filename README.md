# Firmware-for-AppScript-GoogleSheet
## Arduino_IDE
- used to upload into the microcontroller Adafruit ESP32 via Arduino IDE program. The library files are available in **Library Manager** menu.
- This was designed for connected to the university Eduroam wifi, SO **USERNAME** and **PASSWOARD** need to be changed
- **ScriptURL** is from your Google Sheet

## GoogleAppScript
- used for set up the Google Sheet
